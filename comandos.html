Início Sobre Status do Bot

Informações / Comandos

- Usa-se o /play para adicionar músicas.

- Usa-se o /pause para pausar sua música.

- Usa-se o /playnext para tocar a próxima música da lista de reprodução.

- Usa-se o /loop para que sua música toque sequencialmente.

- Usa-se o /volume para alterar o volume da música.

- Usa-se o /clear limpa a fila de reprodução.

- Usa-se o /queue para mostrar a lista de reprodução.

- Usa-se o /ping para mostrar o ping do Bot.

Para mais comandos, digite o /help.
